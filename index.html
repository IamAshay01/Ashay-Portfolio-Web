To add files to your live website, upload them manually to your GitHub repository:

  1. Go to your GitHub repository: IamAshay01/Ashay-Portfolio-Web

  3. Drag and drop your photos/videos into the upload area
  4. Commit the changes with a message like "Add new portfolio files"
  5. Wait a few minutes for GitHub Pages to update the live site
